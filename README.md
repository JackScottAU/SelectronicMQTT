Required appsettings.json configuration:

```
  "Selectronic": {
    "Email": "",
    "Password": "",
    "SystemNumber": ""
  },
  "Mqtt": {
    "Hostname": "",
    "Username": "",
    "Password": ""
  }
```

The system number is the end portion of the URL (likely to be a number with a 1-3 digit length).
